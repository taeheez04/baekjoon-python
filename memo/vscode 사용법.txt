crtl + F : 현재 파일에서 단어 찾기 // 마우스 클릭 후 crtl + F해도 그 기능 실행
crtl + H : 현재 파일에서 단어 바꾸기 (오타 수정)
crtl + F2 : 커서에 있는 단어 바로 찾기 / 그 후 변경도 가능 (변수명 바꿀 때)

crtl + shift + F : 프로젝트 전체에서 단어 찾기

crtl + K > F : 열려있는 폴더 닫기
crtl + shift + N : 새 창 열기
